## OpenGOO pre-Alpha2

A free and open clone of [World of Goo](http://www.worldofgoo.com/)

* [Home page](http://mandarancio.github.com/OpenGOO/)

## DEPENDENCIES

* Qt5 and Qt5 opengl module
* Gcc
* QMake
* Make
