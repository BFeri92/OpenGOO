OpenGOO by Mandarancio

A free clone of World Of Goo

This is a simple personal project for learn box2d (a 2d physic engine). The project is of course free and open. I use qt and open gl for graphic and objects interaction and Box2d for the physic

Dependencies

Box2D >=2.2.1 Qt >= 4.5 OpenGl

Install

qmake && make
