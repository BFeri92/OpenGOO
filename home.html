Started like a simple personal project, OpenGOO has now a little team that believes in its future. The game is just in pre-alpha stage but we are working to publish the first alpha version. We need active contributors!

OpenGOO is not just a clone, we want to include several new concepts starting from a unique and automatic installer on several linux distributions. Here's the full list, but we accept new ideas:

  • Goos with different weights.
  • A playable game menu.
  • A collaborative story.
  • A crash report sender tool with feedback about the fixed bugs.
  • An automatic installer on different linux distributions.

With this in mind, the game entered the Liberated Pixel Cup in the latest months, an art and code competition.
