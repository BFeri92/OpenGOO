Started like a simple personal project OpenGOO has now a little team that believes in the future of this project. The game is just in pre-alpha stage but we want to release the first alpha version for the end of october.

OpenGOO is not just a clone, we want to improve the game with several new concepts starting from a unique and automatic installer on several linux distributions. Here's the full list, but we accept new ideas:

  • Goos with different weights.
  • A playable game menu.
  • A collaborative story.
  • A crash report sender tool with feedback about the fixed bugs.
  • An automatic installer on different linux distributions.

With this in mind, the game entered the Liberated Pixel Cup in the latest months, an art and code competition.
